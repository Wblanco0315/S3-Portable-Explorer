# Project Guidelines

## UI/UX Rules
- **Icons**: Always use `react-icons` (preferably Heroicons or Remix Icons). **Never use emojis** for UI elements as they look unprofessional across different operating systems.
- **Styling**: Use Tailwind CSS for all components. Follow the premium design system established in the layout.
- **Routing**: Use `react-router-dom` for all navigation. Use `NavLink` for sidebar items to maintain active state.

## Tech Stack
- **Framework**: React 19 + TypeScript
- **Runtime**: Bun
- **Styles**: Tailwind CSS v4
- **Icons**: `react-icons`
- **Desktop**: Tauri v2
